Pourquoi un fork et pas un clone ? On ne sait jamais vous voudriez peut être proposer des modifications dans le template du site web original …

Si vous voulez consulter votre site web en local, il faut que vous ayez installer ruby sur votre ordinateur. Ensuite il suffit de se positionner dans le bon répertoire et de lancer un serveur web local avec les commandes suivantes :

bundle exec jekyll serve

Le résultat est visible dans votre navigateur internet à l'url suivante : http://localhost:4000
